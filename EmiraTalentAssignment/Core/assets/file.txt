The task is to write a text search utility, find in file that will allow a user to specify a file at command line and then input search words. The input of the search terms will be single word each the utility will display all occurrences of the search term along with three words before and after the search term from the document to give it a context. The program will then allow the user to enter the next search term. The program will exit on the special search term “letsquit”.
